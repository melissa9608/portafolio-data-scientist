Optimización de Modelos de Seguros para Sure Tomorrow

Este proyecto aborda cuatro objetivos clave para la compañía de seguros Sure Tomorrow, explorando soluciones de machine learning que mejoren la eficiencia en la gestión de clientes y la seguridad de los datos. Los objetivos incluyen la segmentación de clientes para marketing, la predicción de prestaciones de seguros, el análisis de frecuencia de reclamaciones y el enmascaramiento de datos para proteger la privacidad sin afectar la calidad de los modelos.

Hipótesis

  • Similitud de Clientes: Es posible agrupar clientes similares mediante técnicas de k-vecinos, permitiendo segmentar mejor las campañas de marketing.
  • Prestaciones de Seguros: Un modelo predictivo bien entrenado será más efectivo que un modelo aleatorio (dummy) para identificar la probabilidad de que un cliente reciba una prestación.
  • Frecuencia de Prestaciones: Un modelo de regresión lineal puede estimar de manera precisa la cantidad de prestaciones que un cliente podría requerir.
  • Enmascaramiento de Datos: Es factible implementar un algoritmo de enmascaramiento que mantenga la privacidad de los datos sin reducir la calidad de los modelos predictivos.

Metodología

  • Análisis Exploratorio de Datos (EDA): Se realizó una exploración inicial de datos para identificar patrones y relaciones mediante gráficos de pares, aunque la falta de clústeres evidentes impulsó el uso de aprendizaje automático.
  • Clientes Similares (Objetivo 1): Se implementó un modelo de k-Nearest Neighbors (kNN) utilizando métricas de distancia Euclidiana y Manhattan, con y sin escalado de datos, para identificar los clientes más cercanos a uno dado.
  • Modelo Predictivo de Prestaciones (Objetivo 2): Se desarrolló un modelo kNN para predecir la probabilidad de prestación, comparando su rendimiento con un modelo dummy en varias configuraciones de probabilidad. Se midió el rendimiento mediante la métrica F1 para diferentes valores de k.
  • Regresión Lineal para Predecir el Número de Prestaciones (Objetivo 3): Se entrenó un modelo de regresión lineal para estimar la cantidad de prestaciones de seguro esperadas para nuevos clientes.
  • Enmascaramiento de Datos (Objetivo 4): Se creó un algoritmo de ofuscación que protege los datos personales manteniendo el rendimiento de los modelos de machine learning.

Resultados

  • Objetivo 1: Se observó que el escalado de datos afecta significativamente los resultados del modelo kNN, permitiendo una identificación más precisa de vecinos cercanos cuando las variables están en una escala uniforme.
  • Objetivo 2: El modelo kNN escalado superó al modelo dummy, alcanzando F1 Scores de hasta 0.96 para k=1 a 5. El modelo dummy resultó en F1 Scores bajos, confirmando que un modelo entrenado mejora la predicción.
  • Objetivo 3: La regresión lineal proporcionó estimaciones precisas del número de prestaciones esperadas, siendo una herramienta útil para la planificación financiera.
  • Objetivo 4: El enmascaramiento de datos demostró ser efectivo, protegiendo la información sin comprometer la precisión del modelo predictivo.

Conclusiones

Las técnicas de machine learning pueden optimizar considerablemente el análisis de clientes y la predicción de prestaciones en el sector de seguros. El uso de modelos kNN y regresión lineal, junto con el enmascaramiento de datos, proporciona soluciones avanzadas en marketing, finanzas y seguridad de datos.

Descripción del Proyecto

El objetivo principal de este proyecto es pronosticar la tasa de cancelación de clientes para la empresa de telecomunicaciones Interconnect...

Metodología

Limpieza de Datos

  • Se importaron las bibliotecas necesarias y se cargaron los datos.
  • Durante la fase de limpieza, se revisaron los datos en busca de valores faltantes...

Exploración Inicial de los Datos

Se llevó a cabo un análisis exploratorio de datos (EDA) para entender la distribución de las variables...

Ingeniería de Características

Se aplicaron técnicas de ingeniería de características para extraer información valiosa de los datos demográficos...

Modelado y Entrenamiento

Se entrenaron múltiples modelos de machine learning, incluidos:

  • Regresión Logística
  • Árboles de Decisión
  • Random Forest
  • Gradient Boosting Machines (CatBoost)
  • XGBoost
  • Support Vector Machine (SVM)

Evaluación del Modelo

El modelo SVM mostró resultados prometedores con una precisión general (accuracy) del 75.26%...

Conclusiones

La implementación del modelo permitirá a Interconnect identificar proactivamente a los clientes en riesgo de cancelación...

Descripción del Proyecto

Este proyecto tiene como objetivo desarrollar un modelo que analice el comportamiento de los clientes y recomiende uno de los nuevos planes de la compañía móvil Megaline: "Smart" o "Ultra". La compañía busca reemplazar planes heredados y mejorar la satisfacción del cliente a través de recomendaciones personalizadas. Se trabajó con datos de consumo de llamadas, mensajes, minutos de uso y datos móviles para construir un modelo que clasifique a los clientes en los dos planes mencionados.

Metodología

1. Preparación de los Datos

  • Se importaron las librerías necesarias (pandas, sklearn) y se cargó el dataset.
  • El dataset contiene cinco columnas: calls, minutes, messages, mb_used y is_ultra (0 para "Smart" y 1 para "Ultra"), lo que permite abordar el problema como una clasificación binaria.

2. Segmentación de Datos

  • La variable objetivo es is_ultra..
  • Se dividieron los datos en un conjunto de entrenamiento (60%), validación (20%) y prueba (20%) para evaluar y ajustar el modelo sin riesgo de sobreajuste.

3. Entrenamiento de Modelos

    Se investigaron tres algoritmos de clasificación: Regresión Logística, Árbol de Decisión y Bosque Aleatorio. Regresión Logística: Se probaron los solvers liblinear y lbfgs. La mejor precisión fue 0.72 con liblinear. Árbol de Decisión: Se ajustó el hiperparámetro max_depth, obteniendo una precisión de 0.76 con max_depth=3. Bosque Aleatorio: Este modelo ofreció los mejores resultados con n_estimators=100, alcanzando una precisión de 0.79 en el conjunto de validación.

4. Evaluación de Modelos

Los modelos fueron evaluados con el conjunto de prueba, y el Bosque Aleatorio resultó ser el mejor modelo con una precisión de 0.79, superando el umbral de exactitud de 0.75.

4. Prueba de Cordura

Se realizó una prueba de cordura para validar que el modelo final, Bosque Aleatorio con n_estimators=100, generaliza correctamente y no está sobreajustado.

Conclusiones

El modelo de Bosque Aleatorio con n_estimators=100 fue seleccionado por su alta exactitud en la clasificación entre los planes Smart y Ultra, demostrando ser efectivo y confiable para recomendar el plan adecuado según el comportamiento del cliente.

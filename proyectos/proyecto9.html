Predicción del Valor de Autos Usados para Rusty Bargain con Machine Learning

Este proyecto se centra en la predicción del valor de mercado de vehículos usados para la empresa Rusty Bargain, utilizando un conjunto de datos de coches usados con información variada como tipo de vehículo, marca, modelo, tipo de combustible y transmisión, entre otros factores. La intención es crear un modelo que permita estimar de manera precisa el valor de un vehículo basándose en sus características y así mejorar la estrategia de compra-venta de la empresa.

hipótesis

Este proyecto se centra en la predicción del valor de mercado de vehículos usados para la empresa Rusty Bargain, utilizando un conjunto de datos de coches usados con información variada como tipo de vehículo, marca, modelo, tipo de combustible y transmisión, entre otros factores. La intención es crear un modelo que permita estimar de manera precisa el valor de un vehículo basándose en sus características y así mejorar la estrategia de compra-venta de la empresa.

Herramientas y Tecnologías Utilizadas

  • Python: para todo el procesamiento y modelado de datos.
  • pandas: limpieza y manipulación del dataset.
  • scikit-learn: entrenamiento y evaluación de modelos de regresión lineal, árbol de decisión, y bosque aleatorio.
  • CatBoost: entrenamiento y optimización de un modelo avanzado de boosting.
  • StandardScaler: para escalar características numéricas.
  • Matplotlib & Seaborn: visualización de métricas y resultados del modelo (opcional para gráficos adicionales).

Flujo de Trabajo

  1. Preparación de Datos: se realizó un tratamiento de valores nulos y duplicados, y se eliminaron columnas irrelevantes como last_seen, number_of_pictures, postal_code, y date_crawled. Se aplicó codificación One-Hot para las variables categóricas.
  2. División de Datos: el conjunto de datos se dividió en conjuntos de entrenamiento (75%) y validación (25%) para evaluar el rendimiento de los modelos de manera justa.
  3. Entrenamiento y Comparación de Modelos: se entrenaron varios modelos para comparar su rendimiento en la predicción de precios:
    • Regresión Lineal: modelo base de comparación para establecer un punto de referencia.
    • Árbol de Decisión: se optimizó la profundidad del árbol para mejorar la precisión.
    • Bosque Aleatorio: se experimentó con varios valores de n_estimators para optimizar el rendimiento.
    • CatBoost: modelo de boosting avanzado, ajustado en cuanto a profundidad y número de iteraciones para maximizar la precisión.
  4. Evaluación de Rendimiento: se compararon los modelos utilizando métricas como Error Cuadrático Medio (MSE), Error Absoluto Medio (MAE), R² y tiempos de entrenamiento y predicción.

Comparación de Modelos

  • CatBoost: obtuvo el menor MSE y el mayor R², indicando que es el modelo más preciso y eficiente en términos de capacidad de predicción. A pesar de ser más lento en entrenamiento, ofrece un buen equilibrio entre tiempo y precisión.
  • Bosque Aleatorio: proporciona un rendimiento sólido en precisión y velocidad, siendo una opción alternativa si se busca reducir el tiempo de entrenamiento ligeramente.
  • Árbol de Decisión: es el modelo más rápido en términos de entrenamiento y predicción, aunque sacrifica precisión. Es adecuado en contextos donde la velocidad es prioritaria.
  • Regresión Lineal: aunque es rápida, su precisión es considerablemente menor en comparación con los otros modelos.

Conclusiones

Después de comparar varios modelos de machine learning, el modelo CatBoost resultó ser el mejor en términos de precisión y capacidad de generalización, gracias a su baja tasa de error (MSE) y alto R². Esto significa que el modelo puede predecir el valor de mercado de coches usados con un alto grado de confianza, proporcionando una herramienta valiosa para Rusty Bargain en su estrategia de compra y venta de vehículos.

Este proyecto demuestra cómo los modelos de machine learning pueden mejorar los procesos de negocio al proporcionar estimaciones precisas basadas en datos históricos. Para futuras iteraciones, se podría explorar el uso de modelos adicionales y optimizar los hiperparámetros mediante búsqueda en cuadrícula para mejorar aún más la precisión.

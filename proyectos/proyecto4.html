Descripción del Proyecto

Este proyecto tiene como objetivo desarrollar un modelo de predicción para identificar la probabilidad de cancelación de clientes en el sector de telecomunicaciones. La cancelación de clientes representa una pérdida significativa para las empresas, y la retención es clave para la sostenibilidad del negocio. Este análisis proporciona insights sobre los factores que inciden en la pérdida de clientes, y plantea recomendaciones para mitigar este riesgo.

Hipótesis

Se planteó la hipótesis de que los clientes que presentan comportamientos específicos, como un uso reducido de servicios adicionales, quejas recurrentes o largas duraciones en el servicio al cliente, tienen mayor probabilidad de cancelar su suscripción. Además, se consideró que los datos demográficos y la duración de la relación con la empresa influyen en la cancelación.

Metodología

1. Análisis Exploratorio de Datos (EDA):

Se exploraron las características de la base de datos y se identificaron patrones relevantes y correlaciones entre variables.

2. Preprocesamiento de Datos

Se realizó una limpieza de datos, imputación de valores nulos y codificación de variables categóricas para optimizar la entrada al modelo.

3. Selección de Características

Se aplicaron métodos de selección de características como el análisis de correlación y técnicas de reducción de dimensionalidad para identificar las variables más relevantes.

4. Modelado Predictivo

Se entrenaron varios modelos de machine learning, incluidos modelos de regresión logística, árboles de decisión, y un modelo de bosque aleatorio (Random Forest), optimizado mediante ajuste de hiperparámetros para maximizar la precisión y reducir el sobreajuste.

5. Validación del Modelo

La eficacia del modelo se evaluó utilizando métricas como precisión, sensibilidad, especificidad y la métrica F1, empleando validación cruzada para garantizar la estabilidad y confiabilidad de los resultados.

Conclusiones

Los resultados indicaron que ciertos factores, como la cantidad de quejas registradas, el tiempo total con la empresa y el uso de servicios específicos, son indicadores fuertes de cancelación. Las empresas de telecomunicaciones pueden focalizar estrategias de retención en los clientes con alto riesgo de cancelación mediante personalización de ofertas y mejoras en el servicio al cliente.

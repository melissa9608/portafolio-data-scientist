Predicción de Pedidos de Taxis en Tiempo Real para Sweet Lift Taxi

Este proyecto tiene como objetivo desarrollar un modelo de aprendizaje automático para predecir la cantidad de pedidos de taxis en el aeropuerto para la próxima hora. Utilizando datos históricos, se busca implementar una solución eficaz para predecir la demanda de taxis, lo que ayudará a optimizar los recursos y mejorar el servicio para los clientes.

La predicción se realiza utilizando un conjunto de técnicas de Machine Learning y análisis de series temporales, como el modelo SARIMA y otros modelos de regresión como Random Forest, Árbol de Decisión y Regresión Lineal.

Herramientas y Tecnologías

  • Pandas: Manipulación y limpieza de datos.
  • Scikit-learn: Modelos de Machine Learning (Regresión Lineal, Árbol de Decisión, Random Forest).
  • SARIMAX: Modelo SARIMA para series temporales.
  • Statsmodels: Análisis de series temporales y descomposición estacional.
  • Matplotlib: Visualización de resultados.

Metodología

La metodología seguida incluye los siguientes pasos:

  1. Preprocesamiento de datos: Limpieza y remuestreo de datos a intervalos de una hora.
  2. Análisis exploratorio de datos: Estudio de las series temporales mediante la descomposición estacional.
  3. Generación de características adicionales: Creación de nuevas variables basadas en las características temporales.
  4. Entrenamiento y evaluación de modelos: Implementación de diferentes algoritmos de Machine Learning, incluyendo regresión lineal, árboles de decisión, random forest y el modelo SARIMA.
  5. Visualización de resultados: Comparación y análisis visual de las predicciones realizadas por los modelos.

Resultados

Los modelos fueron evaluados utilizando la métrica de Error Cuadrático Medio de la Raíz (RECM), con el siguiente desempeño:

  • Regresión Lineal: RECM = 53.41
  • Árbol de Decisión: Mejores Hiperparámetros: max_depth = 3, RECM = 52.60
  • Random Forest: Mejores Hiperparámetros: n_estimators = 40, RECM = 46.78
  • SARIMA: RECM = 43.64 (Mejor desempeño)

El modelo SARIMA, que maneja patrones estacionales y no estacionarios, se destacó como el más preciso y alcanzó el objetivo de un RECM inferior a 48.

Conclusiones

El objetivo del proyecto fue construir un modelo que predijera la cantidad de pedidos de taxis en la próxima hora. Se probaron diferentes enfoques, incluyendo varios modelos de aprendizaje automático y un modelo SARIMA para series temporales.

Los modelos de Machine Learning, como la Regresión Lineal, el Árbol de Decisión y el Random Forest, no alcanzaron el objetivo de RECM < 48. Sin embargo, el modelo SARIMA fue el que mejor desempeño tuvo, alcanzando un RECM de 43.64, cumpliendo con los requisitos de precisión establecidos.

El modelo SARIMA es el más adecuado para este tipo de problemas, ya que es capaz de manejar patrones estacionales y no estacionarios, lo que es común en los datos de pedidos de taxis, que muestran fluctuaciones según la hora del día y el día de la semana.

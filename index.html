Predicción de Pedidos de Taxis en Tiempo Real para Sweet Lift Taxi

En este proyecto de series temporales, desarrollo un modelo de machine learning para predecir la demanda de taxis en el aeropuerto para la próxima hora, optimizando la disponibilidad de vehículos para Sweet Lift Taxi. Usando datos históricos, aplico análisis de series temporales y técnicas de descomposición estacional para capturar patrones. Tras comparar varios modelos, incluyendo SARIMA y Random Forest, alcanzo un RECM de 43.64, cumpliendo con el objetivo de precisión. Visualizaciones detalladas complementan el análisis, mostrando la efectividad de las predicciones y ayudando a mejorar la planificación operativa.
